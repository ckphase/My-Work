Budget Tracking Tools


What is budget tracking form? The Budget Tracking Form is the form where you can add all the information about your expences for example amount (the money that was spent or added), category (it contains the list of many categories like salary, education and basic needs etc.), date, description and type weather it was a expence or salary. To go to the form click the internal link given below named as Add Income/Expences.

What is My Budget Summary? My budget summary is a table that contains various columns that contain each and every information about the entries made inside the form. This table shows all the columns like expence, amount, description and date etc. Also edit and delete button to delete or edit any entry option is also there in the action column where you can delete or edit repectively any entry that was made in the table using the form.
